# Tanishq Mishra - AI Engineer Portfolio

This repository contains the source code for my personal portfolio website. It serves as a central hub for my academic background, technical capabilities, and project experience in the fields of artificial intelligence, large language models, and scalable backend architecture.

## Overview

The portfolio is designed with a premium, terminal-inspired aesthetic that highlights my focus on building AI systems rather than simple demonstrations. It includes sections detailing my core architecture (about me), a comprehensive skill matrix, and operational logs (internship and project experience).

## Technical Stack

The site is built as a single-page application focused on performance and dynamic styling:

- Framework: React with Vite for fast, optimized builds
- Styling: Tailwind CSS v4 with custom CSS Houdini animations
- Icons: Lucide React
- Typography: A blend of modern sans-serif (Syne), bold display (Bebas Neue), and monospace (DM Mono) fonts for a highly technical aesthetic
- Language: TypeScript

## Key Features

- Dynamic Theme System: A custom light and dark mode toggle seamlessly integrated into the React application, driving extensive Tailwind variant styling.
- Hardware-Accelerated Animations: Complex background geometries and particle systems utilizing pure CSS math and properties for maximum rendering efficiency.
- Terminal Simulation: An interactive React component simulating a booting system terminal to reinforce the engineering theme.
- Interactive Contact Protocol: A specialized contact form that parses inputs and automatically delegates message transmission to the user's default email client.

## Running Locally

To run this project on your local machine, follow these steps:

1. Ensure you have Node.js installed on your system.
2. Clone this repository and navigate to the project directory.
3. Install the required dependencies:
   npm install

4. Start the Vite development server:
   npm run dev

5. Open the provided localhost link (typically http://localhost:5173) in your browser.
